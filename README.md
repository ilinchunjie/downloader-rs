# downloader-rs
a simple file downloader for rust

# Support Features
- [x] multithreading/chunk/resumable download
- [x] download in memory
- [ ] bandwidth limit
- [x] thread count limit
- [ ] retry times after failure
- [ ] md5/xxHash file verification
